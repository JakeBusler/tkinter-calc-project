Tkinter Calculator: Mid Term Project
A robust, secure, and visually snazzy calculator built with Python's Tkinter library. This application was developed as a mid-term project focusing on clean GUI design, responsive layouts, and safe mathematical evaluation.

Features
Secure Math Evaluation: Utilizes Python's ast (Abstract Syntax Tree) module to safely parse and evaluate user input, entirely avoiding the security vulnerabilities associated with eval().

Responsive GUI Layout: Built using Tkinter's .grid() manager, ensuring buttons and displays scale appropriately.

Quality of Life Bindings: Full keyboard and numpad integration for rapid data entry without needing a mouse.

Extended Functionality: Goes beyond basic addition with support for subtraction, multiplication, division, percentage conversions, and positive/negative sign toggling.

Custom Styling: Features a custom "Frosted Wild-Berry" inspired color palette with dynamic hover and click states.

Setup & Execution
No external dependencies or bundled packages are required. Simply ensure you have Python 3 installed.

To launch the calculator, run the following command in your terminal:

Bash
python calculator.py
